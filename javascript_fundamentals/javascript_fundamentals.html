JavaScript Basics

  • script is the necessary tag to place JavaScript in the page
  • To import a JS file, use the src attribute in the tag
  • console.log(varName) to request information from the console about a variable
  • console.log(varName) also serves to display information from the code to the console
  • To concatenate strings, use the + followed by a single quote
  • 'use strict' - the strict mode of JS goes at the beginning of files and helps us program in a better way
  • VAR varName = something to create variables, use the reserved word var although LET is also used
  • The difference between var and let is that let declares or modifies the variable within a code block "limited scope" and var allows updating regardless of whether it is in a block or not "declares it globally"
  • const constant is like a variable but without being able to modify the data at any point in the program

JavaScript Concepts Explained

  • var: Declares a variable globally or locally to an entire function regardless of block scope
  • let: Declares a block-scoped local variable, optionally initializing it to a value
  • const: Creates a read-only reference to a value, block-scoped like let
  • Data Types: JavaScript has various data types including:
    • Numbers: Integer or floating-point (e.g., 44, 40.5)
    • Strings: Text enclosed in quotes (e.g., "hello")
    • Booleans: true or false values

Conditionals

if...else statements
Executes different code blocks based on specified conditions

switch statement
Evaluates an expression and executes code blocks based on matching cases

Loops

for loop
Creates a loop with three optional expressions

while loop
Creates a loop that executes as long as the condition is true

do...while loop
Executes once before checking the condition

break
Terminates the current loop or switch statement
